# Get-vRSLCMRole

## Synopsis

Retrieve VMware Aria Suite Lifecycle roles.

## Syntax

``` PowerShell
Get-vRSLCMRole
```

## Description

The `Get-vRSLCMRole` cmdlet retrieves a list of VMware Aria Suite Lifecycle roles.

## Examples

### Example 1

``` PowerShell
Get-vRSLCMRole
```

This example retrieves a list of VMware Aria Suite Lifecycle roles.
