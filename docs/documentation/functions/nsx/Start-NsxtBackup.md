# Start-NsxtBackup

## Synopsis

Start the backup of NSX Manager.

## Syntax

```powershell
Start-NsxtBackup
```

## Description

The `Start-NsxtBackup` cmdlet starts the backup of NSX Manager.

## Examples

### Exmaple 1

```powershell
Start-NsxtBackup
```

This example starts a backup of NSX Manager.
