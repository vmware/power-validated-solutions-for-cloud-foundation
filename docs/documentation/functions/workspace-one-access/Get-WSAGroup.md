# Get-WSAGroup

## Synopsis

Get groups

## Syntax

```powershell
Get-WSAGroup [[-id] <String>] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## Description

The `Get-WSAGroup` cmdlets retrieves the groups in Workspace ONE Access.

## Examples

### Example 1

```powershell
Get-WSAGroup
```

This example retrieves the groups in Workspace ONE Access.
